##

### Overview

Build a production-ready RAG-powered intelligent chatbot for Telecom Egypt that answers
customer queries using the official website as the primary knowledge base. The system must
support multi-lingual processing (Arabic and English), handle Egyptian dialect variations, and allow
users to upload documents for querying.

Scenario
Telecom Egypt receives thousands of customer inquiries daily about services, plans, bills, and
technical support. The company wants to deploy an AI-powered chatbot that can:

1. Answers customer questions using Telecom Egypt official website
2. Handles queries in Arabic, English, and Egyptian dialect
3. Allow users to upload and query additional documents (PDF, DOCX, TXT, HTML, Images)
4. All responses must be grounded and bounded by the Telecom Egypt website
5. Provides source citations for all answers
6. Delivers a professional chat interface
